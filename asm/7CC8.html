7CC8: Make a teacher wait for EINSTEIN to finish speaking
The address of this interruptible subcommand routine is placed into bytes 0x69 and 0x6A of a teacher's buffer by the routine at 7CAF.
Input
H Teacher's character number (0xA3-0xA6)
Output
7CC8 LD A,($7FC9) 7FC9 holds the lesson signal flags
7CCB RLCA Is EINSTEIN still speaking?
7CCC RET C Return if so
7CCD JP $62A4 Terminate this interruptible subcommand
