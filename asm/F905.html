F905: Collect the identifier for the next lesson
Used by the routine at 66E6. Returns with A holding the identifier (224-255) for the next lesson. If the next lesson is PLAYTIME, one of the 'special' PLAYTIMEs (243, 244 or 245) may be chosen in place of the main timetable entry.
Input
E Lesson number (0-63)
Output
F905 LD D,$FE Pick up the lesson identifier (224-255) from the main timetable
F907 LD A,(DE)
F908 CP $F3 Is it PLAYTIME (243-255)?
F90A RET C Return if not
F90B LD A,($7FEA) 7FEA holds the game mode indicator
F90E AND A Set the zero flag if we're in demo mode
F90F LD A,(DE) Restore the lesson identifier to A and return if we're in demo mode
F910 RET Z
F911 CALL $61A1 A=random number
F914 JP $6E33 Select a 'special' PLAYTIME occasionally
