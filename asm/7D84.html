7D84: Make a teacher tell the kids to sit down
Used by the teachers' command lists 0x80, 0x82, 0x84 and 0x86. Makes the teacher tell the kids to sit down; also makes MR CREAK reveal his safe combination letter if he can see his year of birth written on the board.
Input
H Teacher's character number (0xA3-0xA6)
7D84 CALL $F6D7 Make MR CREAK reveal his safe combination letter if appropriate; also set BC=F1C1 (SILENCE! OR I'LL CANE THE LOT OF YOU)
7D87 LD A,H A=teacher's character number (0xA3-0xA6)
7D88 SUB $A3 Is it MR WACKER?
7D8A JR Z,$7D98 Jump if so
7D8C LD C,$59 BC=F159 (RIGHT! SIT DOWN MY LITTLE CHERUBS)
7D8E DEC A Is it MR ROCKITT?
7D8F JR Z,$7D98 Jump if so
7D91 LD C,$7B BC=F17B (COME ON CHAPS - SETTLE DOWN)
7D93 DEC A Is it MR WITHIT?
7D94 JR Z,$7D98 Jump if so
7D96 LD C,$97 It must be MR CREAK; set BC=F197 (BE QUIET AND SEATED YOU NASTY LITTLE BOYS)
7D98 CALL $7D26 Make the teacher speak
7D9B JP $62A8 Move to the next command in the command list
