67D8: Get input from the Kempston joystick if it's in use
Used by the routine at 673A. Returns with the zero flag set if there was no joystick input. Otherwise returns to the caller of 673A with the zero flag reset and the ASCII code of the equivalent keypress in A.

67DB LD A,($681F) A=0x6B ('k') if the Kempston joystick is being used, 0x00 otherwise

67E4 RET Z Return with the zero flag set if there was no input

67F8 RET NZ Return if the joystick was moved left
